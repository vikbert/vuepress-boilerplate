## Install

```bash
 npm install pixss
```

Then use the css files from `node_modules/pixss`

```bash
├── dist
│   └── styles
│       ├── pixss.css
│       ├── pixss.css.map
│       ├── pixss.min.css
│       └── pixss.min.css.map
```
